# Expense Tracker

A modern Expense Tracker application built with React.js that helps users manage their finances by tracking income and expenses in real time. The application provides an intuitive interface for monitoring transactions, calculating balances, and organizing financial records efficiently.

## Features

* Add income and expense transactions
* Real-time balance calculation
* Track total income and total expenses
* Delete transactions instantly
* Responsive user interface
* Dynamic state management using React Hooks
* Fast performance with Vite
* Deployed on Vercel

## Technologies Used

* React.js
* JavaScript (ES6+)
* Vite
* CSS3
* Vercel

## Installation & Setup

1. Clone the repository:

```bash
git clone <repository-url>
```

2. Navigate to the project directory:

```bash
cd expense-tracker
```

3. Install dependencies:

```bash
npm install
```

4. Start the development server:

```bash
npm run dev
```

5. Open your browser and visit:

```text
http://localhost:5173
```

## How It Works

* Users can add income or expense transactions.
* Each transaction updates the total balance automatically.
* Income and expense summaries are calculated dynamically.
* Transactions can be removed whenever needed.
* React state management ensures instant UI updates.
